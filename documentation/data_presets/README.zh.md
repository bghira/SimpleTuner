# 数据集配置预设

这里提供了 Hugging Face Hub 上多个大规模数据集的配置细节，帮助你快速上手。

> **提示：** 对于大型正则化数据集，可以使用 `max_num_samples` 将数据集限制为确定性的随机子集。详见 [DATALOADER.md](../DATALOADER.md#max_num_samples)。

要新增预设，请使用[此模板](../data_presets/preset.md)提交新的 pull request。

- [bghira/photo-concept-bucket](../data_presets/preset_pexels.md)
- [Midjourney v6 520k](../data_presets/preset_midjourney.md)
