WebUI Preferences

Adjust the appearance and behaviour of the training studio interface.

Theme

Switch between available UI themes. Changes apply instantly.

Event Stream

When disabled, event updates only refresh when triggered manually. Longer intervals reduce background activity.

Configuration Saves

When enabled, configuration saves keep only values that differ from their defaults without asking each time. Disable this to review the save options before writing a config.

Documentation Links

When enabled, external link icons appear next to dataloader form fields linking to documentation. Disable this if you prefer a cleaner interface without the link indicators.

Cloud Training

Enabled by default. Uncheck to hide the Cloud tab from the sidebar.

Public URL for receiving Replicate training events (e.g., via ngrok or cloudflared).
Controls whether to ask for confirmation before uploading training data to cloud providers.

When enabled, the Cloud tab provides access to Replicate cloud training and job management. Set REPLICATE_API_TOKEN environment variable to authenticate. Docs

Personal API Token

Optionally provide your own Replicate API token. If set, it will be used instead of the global token.

User Management

Enabled by default. When auth is configured, only admins see this tab.

Git Mirror

Git preferences apply to the configs directory. Enable the tab to expose the Git Mirror controls.
